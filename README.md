# Weather App

A small client-side weather viewer that fetches current conditions for a selected city and displays temperature, humidity, wind, and a condition icon.

## Features
- Select a city from the dropdown and fetch up-to-date weather data.
- Displays: temperature, feels-like, humidity, wind speed and gusts, and a weather icon/description.
- Simple, responsive UI with custom styling.

## Files
- `Index.html` — app HTML shell
- `Styles.css` — styling and layout
- `Script.js` — fetches data and updates the UI

## Usage
1. Open `Index.html` in your browser (or serve the folder with a simple static server).
2. Choose a city from the dropdown and click "Get Weather".

Notes:
- The app uses the free proxy API at `https://weather-proxy.freecodecamp.rocks/api/city` to retrieve OpenWeather-like responses.
- If you plan to publish this repository, consider replacing the proxy with your own API key or server.
